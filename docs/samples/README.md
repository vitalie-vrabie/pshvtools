# Samples

This folder contains usage examples and sample scripts demonstrating typical PSHVTools workflows.

## Available Samples

- `fix-vhd-acl.samples.ps1` - Example usage of ACL repair utility
- `hvbak.samples.ps1` - Example backup run
- `hvclone.samples.ps1` - Example VM clone operation
- `hvcompact.samples.ps1` - Example VHD compaction
- `hvrecover.samples.ps1` - Example orphaned VM recovery

Refer to the `samples/` folder in the repository for the actual script files.
